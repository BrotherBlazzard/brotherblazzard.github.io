Lesson 01 - Introduction to the Course and C#

Module 1: The C# Language and Visual Studio IDE

Overview

Welcome to the introductory lesson on ASP.NET programming using C#. We begin this course by laying a foundation of the C# programming language and setting up the development environment. In this lesson we discover the relevancy of the C# language within the development industry, install the Microsoft Visual Studio integrated development environment (IDE), and leverage our basic data processing knowledge with the C# programming language. In addition, we will begin to form development teams by introducing ourselves and participating in the weekly, virtual lab class.

Objectives

  • Develop and demonstrate the ability to collaborate in the online learning environment
  • Relate the course syllabus and outcomes to the scheduled lessons with the semester time frame
  • Setup the Microsoft Visual Studio integrated development environment
  • Establish a community of learners with common goals

Instructions

Download and install Visual Studio Community. We will be using this edition throughout the course in examples and activities.

Please note that Visual Studio Enterprise is also an acceptable install but not recommended. Both of these editions are free to BYU-Idaho students. We will not be using Microsoft WebMatrix during the course. You will discover referenced and other resources and tutorials that reference Microsoft WebMatrix, however, we will be working in Visual Studio exclusively for all the lessons in the course. Visual Studio Community will support everything we need to complete projects in this course.

You will need to have administrator credentials to install Visual Studio.

  1. Download Visual Studio Community: https://www.visualstudio.com/
  2. Install Visual Studio using the following settings: Use Default Settings which includes C# and VB, Web and Desktop features.
  3. Configure the IDE to match your style
  4. Add a new Text File in the C# and name the file using your last name .txt. Within Visual Studio, you can use the menu File -> New -> File to get to the appropriate dialog box. Make sure you have selected Visual C# as the installed type with Text File. The text file should show line numbers and contain your full name, current location, and operating system.

OS X users will need to install and run Visual Studio Community in Windows using Boot Camp, VirtualBox, VMware Fusion, or Parallels Desktop installation on your Mac. Microsoft Windows is free through the department provided Microsoft On The Hub student account.

BYU-Idaho Learning Model.

The purpose of this assignment is for you to take the opportunity to teach someone or a small group outside of the class the principles and concepts that you learned to date in this course. The best option is for you to be able to teach the same group each week. Focus on concepts that you can correlate with other coursework within your chosen major and degree plan and/or to practical, work-based experiences. You will spend ~10-15 minutes teaching. You will be asked to report on the individual or group you taught and the concept(s) that you covered. In addition, you will report on any questions or feedback from the individual or group that you taught and reflect on insights that came to you from this teaching opportunity.
