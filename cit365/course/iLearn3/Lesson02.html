Lesson 02 - Using C# and Visual Studio

Module 1: The C# Language and Visual Studio IDE

Overview

In this lesson we discover the relevancy of the C# language within the development industry, use the Microsoft Visual Studio integrated development environment (IDE) to build a simple windows application, and leverage our basic data processing knowledge with the C# programming language. We will investigate C# features, syntax, variables, identifiers, data types, operators, decision blocks, loops, and general program flow. By the end of the lesson you will be able to converse with others about these concepts and be able to explain a simple console application written in C#.

Objectives

  • Leverage and apply fundamental programming concepts and structures in the C# language
  • Use the Visual Studio integrated development environment at the basic, operational level
