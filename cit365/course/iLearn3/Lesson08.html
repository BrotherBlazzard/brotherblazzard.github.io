Lesson 08 - Layout, UI, UX, and Helpers

Module 2: ASP.NET Web Pages with Razor

Overview

In this lesson we continue to learn the ASP.NET Web Pages framework by working on user interaction through form processing, form validation, multimedia, and overall user experience concepts. In addition, we investigate a handful of the available helpers in ASP.NET. which are built-in or custom-made components that help simplify common web programming tasks such as the WebGrid for data display and the Chart Helper for charting images. The programming exercise in this lesson focuses on the module's team project which is due at the end of the next lesson.

Objectives

  • Create pages with consistent layouts and themes using ASP.NET Web Pages tools
  • Process forms including validating user input
  • Customize site-wide behavior
  • Work with multimedia objects
  • Understand the rule of modularized 'Helper' libraries and use them in application development.
