Lesson 07 - Working with Database Data

Module 2: ASP.NET Web Pages with Razor

Overview

We continue our work with ASP.NET Web Pages by working with data which is the foundation of most web applications and the dynamic nature of the content of those sites. During this lesson we review the basic structure of databases, build a database using SQL Server Compact Edition, and build an application that views, adds, edits, and deletes records through a form interface. The lesson is built around an online tutorial that provides an overview of all of these common data activities and includes work on connectivity, validation, rich controls (helpers), and data manipulation through SQL commands.

Objectives

  • Leverage database knowledge in the ASP.NET Web Pages environment
  • Run SQL commands in ASP.NET Web Pages
  • Display, add, update, and delete data from a database
  • Customize controls and helpers to leverage development cycles
