CIT 160 - Introduction to Programming

Syllabus

Description

This course is an introduction to programming logic and design using JavaScript as the learning languages. The course covers programming logic, common control structures, functions, arrays, and language specific concepts.

Topics

  • Basic Computer Hardware and Systems
  • Algorithm Development and Debugging
  • Variables and Data Types
  • Conditions and Compound Conditions
  • Sequential Structures
  • Selection Structures
  • Repetition Structures
  • Functions
  • Arrays

Objectives

Each student will develop fundamental knowlege and skills in computer programming and technology by doing the following:

  1. Understand basic computer hardware architecture, including the function of components.
  2. Understand basic software architecture, including operating systems and applications.
  3. Be able to create simple, valid HTML pages with basic CSS
  4. Write and evaluate compound boolean expressions.
  5. Use segmentation (divide & conquer) to design a solution for a simple programming problem.
  6. Trouble shoot simulated real world problems and incorrect computer code.
  7. Develop and debug a JavaScript computer program using the three basic programming control structures (sequence, selection, repetition), functions, and arrays.
